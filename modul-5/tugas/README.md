# BLACK IRON DEALERSHIP

Website rental motor premium berbasis Vite + Tailwind CSS v4.

## Tech Stack
- Vite
- Tailwind CSS v4
- Vanilla JavaScript

## Cara Menjalankan

Install dependencies:
```bash
npm install
```

Jalankan dev server:
```bash
npm run dev
```

Build production:
```bash
npm run build
```
